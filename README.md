# Blogicum

## Описание проекта

**Blogicum** — это веб-приложение-блог, построенное на Django. Проект представляет собой простой блог-платформу для публикации постов с поддержкой категорий. В качестве демонстрационного контента используются отрывки из дневника Робинзона Крузо.

### Основной функционал

- **Главная страница** — отображение списка всех постов блога
- **Детальная страница поста** — просмотр полного содержимого отдельного поста
- **Фильтрация по категориям** — просмотр постов по выбранной категории
- **Статические страницы** — страницы "О проекте" и "Правила"
- **Адаптивный дизайн** — использование Bootstrap для создания современного и отзывчивого интерфейса

## Технологический стек

### Backend
- **Django 3.2.16** — веб-фреймворк для Python
- **SQLite** — база данных (по умолчанию)

### Frontend
- **Bootstrap** — CSS-фреймворк для создания адаптивного дизайна
- **Django Templates** — система шаблонов Django

### Инструменты разработки и тестирования
- **pytest 7.1.3** — фреймворк для тестирования
- **pytest-django 4.5.2** — плагин для интеграции pytest с Django
- **flake8 5.0.4** — линтер для проверки стиля кода
- **flake8-docstrings 1.7.0** — проверка документации кода
- **pep8-naming 0.13.3** — проверка соответствия именования PEP 8

## Установка и развертывание

### Предварительные требования

- Python 3.9 или выше
- pip (менеджер пакетов Python)
- Git (для клонирования репозитория)

### Пошаговая инструкция

1. **Клонируйте репозиторий:**
   ```bash
   git clone <url-репозитория>
   cd django_sprint1
   ```

2. **Создайте виртуальное окружение:**
   ```bash
   python -m venv venv
   ```

3. **Активируйте виртуальное окружение:**
   
   На Windows:
   ```bash
   venv\Scripts\activate
   ```
   
   На Linux/macOS:
   ```bash
   source venv/bin/activate
   ```

4. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

5. **Выполните миграции базы данных:**
   ```bash
   cd blogicum
   python manage.py migrate
   ```

6. **Создайте суперпользователя (опционально, для доступа к админ-панели):**
   ```bash
   python manage.py createsuperuser
   ```

7. **Запустите сервер разработки:**
   ```bash
   python manage.py runserver
   ```

8. **Откройте браузер и перейдите по адресу:**
   ```
   http://127.0.0.1:8000/
   ```

## Настройка переменных окружения

В текущей версии проекта переменные окружения не используются. Однако для продакшн-развертывания рекомендуется вынести чувствительные данные в переменные окружения.

### Рекомендации для продакшена

Создайте файл `.env` в корне проекта `blogicum/` и добавьте следующие переменные:

```env
SECRET_KEY=your-secret-key-here
DEBUG=False
ALLOWED_HOSTS=your-domain.com,www.your-domain.com
```

Затем установите пакет `python-decouple` или `django-environ` и обновите `settings.py` для чтения переменных из `.env` файла.

**Важно:** Файл `.env` должен быть добавлен в `.gitignore` и не должен попадать в систему контроля версий!

## Запуск тестов

Проект использует pytest для тестирования. Для запуска всех тестов выполните:

```bash
cd blogicum
pytest
```

Для более подробного вывода:

```bash
pytest -vv
```

## Проверка кода

Для проверки стиля кода с помощью flake8:

```bash
flake8 blogicum/
```

## Основные URL-маршруты

- `/` — главная страница со списком постов
- `/posts/<id>/` — детальная страница поста
- `/category/<category_slug>/` — посты по категории
- `/pages/about/` — страница "О проекте"
- `/pages/rules/` — страница "Правила"
- `/admin/` — административная панель Django
